ILAIMH I/ECMH Credential — Cost Model
Overview
This dashboard was created by Start Early, in collaboration with ILAIMH. All hours and cost estimates were reviewed and validated by ILAIMH, based on the "true time" it costs to operate the current Credential program model. See here for the primary documentation of cost model estimates. This cost model and "true time" of Credential operation is consistent across both current versions of the program: (1) operated and managed solely by ILAIMH and (2) operated and managed collaboratively by ILAIMH and Erikson. Questions? Reach out to Jay Wade at jwade@startearly.org.
Purpose
This dashboard allows the ILAIMH team to estimate how changes to the Credential model affect the true cost, as they work to refine and sustain its model in the future.
